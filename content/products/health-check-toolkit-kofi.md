## Long description (~1000 chars)

Every Monday morning, run quick-health-check.sh and you'll know
in 5 seconds whether anything's wrong with your servers. The
script checks disk, memory, CPU, recent errors, network
connectivity, uptime, and failed systemd units. Color-coded
output, JSON mode for monitoring integration, proper exit codes
for alerting.

When disk is filling up, disk-analyzer.sh finds the top 10
culprits in seconds. When error patterns spike, log-watcher.sh
alerts via webhook or email.

Plus a printable one-page weekly checklist and a decision tree
for every warning those scripts can throw at you — so you know
exactly what to do when something's red.

12 KB total. MIT licensed. No dependencies. You own the scripts
forever, modify them freely, redistribute as you wish.

Works on: Ubuntu 20.04+, Debian 11+, RHEL 8+, Amazon Linux 2,
and any Linux with bash 4.0+.

Author: 15-year sysadmin, built this after my own mom lost $4,200
to a tech-support scam — because the best monitoring is the kind
you actually do every week.
