We all get on the call together. I do the configuring, your parent sees how it works, you handle any questions they have. **Most parents learn Buddy by watching us set it up.** This is the option that prevents the most follow-up confusion.

## What if my parent is reluctant?

This is the most common concern. A few things that help:

1. **Frame it as "your kids set this up for you"** — not "you need to learn this app." The setup is done by me, not by them.
2. **Show them the screen recording afterward.** Most reluctant parents become enthusiastic users once they see how simple it is.
3. **Start with one feature.** If they only ever use the "Call my daughter" button, that's a huge win.

If your parent outright refuses, the setup call isn't going to help. I'll refund the $29 if that's the case — no questions.
