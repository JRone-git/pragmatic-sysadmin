# Sysadmin Tools

Welcome to the tools section! Here you'll find interactive helpers for common IT and sysadmin tasks:

## 🎮 New Interactive Learning Tools

### [CLI Adventure Troubleshooter](/tools/cli-adventure.html)
Test your sysadmin skills in our interactive CLI adventure game. Solve real-world IT scenarios and earn professional resume lines.

**Features:**
- 10+ realistic IT scenarios across 3 difficulty levels
- Interactive terminal interface with command validation
- Professional resume line generator for each completed scenario
- Gamified learning with points, streaks, and achievements
- Safe environment for practicing command-line skills

**Perfect for:** Junior sysadmins building confidence or experienced professionals testing their troubleshooting skills.

### [Corporate-Speak Incident Report Generator](/tools/corporate-translator.html)
Transform your raw incident reports into professional corporate language. Generate PDF reports with just a few clicks.

**Features:**
- Automatic translation of technical jargon to corporate speak
- Professional report formatting
- PDF and text file download capabilities
- Corporate bingo card showing detected buzzwords
- Example inputs and outputs for guidance

**Perfect for:** Communicating technical issues to non-technical stakeholders or creating formal incident reports.

### [Danger Zone Command Simulator](/tools/command-simulator.html)
Practice dangerous Linux commands in a safe, simulated terminal environment. Learn command effects without destroying anything.

**Features:**
- Safe simulation of destructive commands
- Visual effects and warnings for dangerous operations
- Virtual filesystem with realistic file operations
- Safety scoring system and danger level tracking
- Educational warnings about real-world consequences

**Perfect for:** Training environment for learning about dangerous commands and their effects without risk.
